## 2. 금일지출계획서 구조 (250430_v3 기준)

### 파일 구조
- 시트: 일별 (0402, 0404, ..., 0430) + 종합
- 각 시트: 은행명, 계좌번호, 거래처명, 내역, 금액, 문서번호, 비고
- 추가 컬럼: 계정분류, 거래처약칭, 내역약칭, 합성키

### 하단 요약
- 원화출금합계 / 외화출금합계
- 계좌잔액요약: 기업(019) 기초잔액 + 기업(019)대체(+) − 지출 총금액(-) = 기업(026) 기말잔액

### 예시 (0404)
- 82건, 합계 357,317,025원

- 지로납부 3건 = 8,629,060
